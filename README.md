# Visual Storyteller Hub

A cinematic portfolio website designed for visual storytellers.

## Project info

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

To run this project locally:

1.  **Clone the repository:**
    ```sh
    git clone <YOUR_GIT_URL>
    ```

2.  **Navigate to the project directory:**
    ```sh
    cd visual-storyteller-hub
    ```

3.  **Install dependencies:**
    ```sh
    npm install
    ```

4.  **Start the development server:**
    ```sh
    npm run dev
    ```

## Deployment

You can deploy this project using any standard static site hosting service like Vercel, Netlify, or GitHub Pages.
